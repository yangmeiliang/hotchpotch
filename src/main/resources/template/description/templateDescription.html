说明文档：
基本参数同 idea设置中的 File and code Templates 模板说明
${tableSaveInfo.basePackage}   默认: cn.xxx.xxx
${tableSaveInfo.basePath}   默认: /src/main/java
${tableSaveInfo.entityNameSuffix}   实体类名后缀，默认为空
${tableSaveInfo.mapperNameSuffix}   mapper类名后缀, 默认: Mapper
${tableSaveInfo.daoNameSuffix}   dao类名后缀, 默认: Dao
${tableSaveInfo.daoImplNameSuffix}   daoImpl类名后缀, 默认 DaoImpl
${tableSaveInfo.entityPackageSuffix}   实体类包名后缀, 默认: entity
${tableSaveInfo.mapperPackageSuffix}   mapper包名后缀, 默认: mapper
${tableSaveInfo.daoPackageSuffix}   dao包名后缀, 默认: dao
${tableSaveInfo.daoImplPackageSuffix}   daoImpl包名后缀, 默认: dao.impl
